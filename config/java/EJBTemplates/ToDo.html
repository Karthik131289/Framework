Deploying EJB Application In Orion

To deploy the generated EJB application '${APPNAME}' perform the following steps

 1.Specify the correct path for ORION_DIR in ${APPNAME}\client\setpath.bat.

2.Copy the following statement to ORION_DIR\config\server.xml directory. 

<application name="${APPNAME}"
             path="file:///${CARTDIR}" />

To test the application '${APPNAME}' perform the following steps

1. Start orion.

2. Start ${APPNAME}\client\execute.bat. The message flows and external messages deployed will be displayed. Execute the message flow or external message you want to test.
